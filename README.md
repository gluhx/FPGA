# FPGA

## Наполнение

Здесь будут лежать файлы по курсу _"Системотехника электронно-вычислительных средств, комплексы и сети"_ за 2025-2026 учебный год.

Файлы созданны Глуховым С.А. группа ИУ4-11М.

## Содержание

Помимо первичных заданий и собственных примеро в папка ![tasks](./tasks) лежат решения заданий для получения зачёта:

1. Задание 1 - Фильтр : вычисление последовательности _y<sub>n</sub>=a * y<sub>n-1</sub> + b * x<sub>n</sub>_ - решение лежит в папке ![task_01_filter](./tasks/task_01_filter).

2. Задание 2 - SPI : SPI slave принимает по MOSI данные, а потом выдаёт их параллельным кодом - решение лежит в папке ![task_02_SPI](./tasks/task_02_SPI).

3. Задание 3 (не сделано) - LU cache : если приходит уже лежащее в памяти число, оно перемещается наверх, если его там нет, значит все смещабются вниз. Важно маштабируемость - решение лежит в папке ![task_03_LU_cache](./tasks/task_03_LU_cache).
