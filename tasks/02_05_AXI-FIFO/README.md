# Задание 5 (AXI-FIFO)

## Задача

Сделать AXI очередь с программируемой глубиной(задается глубина ширина битами).

Должна четко сообщать заполнена когда заполнена и пуста когда пуста.

Если она была пустая, потом заполнилась, как только данные полностью закончились, то тут же что она полная (и также что она пуста)

Не писать в полную очередь и читать из пустой(с гарантией).

Но при этом как только не крайнее положение может поттармаживать это нормально.
